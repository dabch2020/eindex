# eIndex - A股情绪指数

基于成交额、融资余额、涨停家数、市场方向四大核心指标的A股市场情绪量化分析工具。

**在线访问：** https://dabch2020.github.io/eindex

## 核心指标

| 指标 | 计算方式 |
|------|---------|
| 全市场换手率 | 沪深两市总成交额 / 总流通市值 |
| 融资余额占比 | 融资余额 / 总流通市值 |
| 涨停家数占比 | 涨停股票数量 / 全市场股票总数 |

每个指标计算过去250个交易日的分位数（0-100），综合情绪指数 = 三个分位数的平均值。

## 交易信号

- 🟢 **买入**：eIndex ≤ 20（市场极度恐慌）
- 🟡 **持有**：20 < eIndex < 80（市场情绪正常）
- 🔴 **卖出**：eIndex ≥ 80（市场极度狂热）

## 数据更新

数据通过 GitHub Actions 每个交易日18:30（北京时间）自动更新。

也可手动运行数据更新脚本：

```bash
pip install akshare numpy
python scripts/update_data.py
```
