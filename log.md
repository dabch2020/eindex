# eIndex 数据获取日志

## 数据缺失记录

### 1. 融资余额数据源
- **状态**: ✅ 已解决
- **历史问题**:
  - `stock_margin_sse`：签名中 `end_date` 默认值硬编码为 `"20230922"`，导致 2023-09-25 之后 592 天融资数据缺失
  - `stock_margin_szse`：接口参数为 `date`（单日查询），不支持 `start_date`
  - `stock_margin_detail_szse`：报错 `Connection reset by peer`
- **解决方案**: 改用 `macro_china_market_margin_sh`（沪市）+ `macro_china_market_margin_sz`（深市）
  - 沪市数据 3867 条（2010-03-31 ~ 2026-03-11）
  - 深市数据 3669 条（2010-03-31 ~ 2026-03-11）
  - 覆盖率：2457 天中 2455 天有数据（99.9%）
- **深市补漏（2026-03-13）**:
  - `macro_china_market_margin_sz` 接口（东方财富转载深交所）存在大段缺失：
    - 2017-03 每周五（10/17/24）零星缺失 3 天
    - 2018-08-01 ~ 08-09 连续缺失 7 天
    - 2023-09-21 ~ 09-27 缺失 5 天
    - 2024-06-14 ~ 2025-06-18 大段断档约 178 天（深交所数据源问题）
  - 共 193 天深市缺失 + 1 天沪市缺失 = 194 天异常
  - 通过 `stock_margin_szse`（深交所单日查询接口）逐日补漏，**已全部补齐**
  - 补漏后数据中半值异常从 194 天降为 0 天
- **剩余缺失**:
  - 2017-03-13：沪深两市数据源均缺失（1天，数据源本身问题）
  - 最新交易日：融资数据通常延迟1个工作日发布，脚本已处理（缺失指标时只用有数据的指标取均值）

### 2. 涨停数据历史覆盖
- **状态**: ✅ 已解决（通达信 880006）
- **历史**: `stock_zt_pool_em`（东方财富）仅保留最近约 1 个月的数据；Tushare Pro token 权限不足
- **解决方案**: 改用 mootdx 库通过通达信行情协议获取 **880006 停板家数** 指标
  - `close` 列 = 涨停家数，`open` 列 = 跌停家数
  - 一次性获取 **2457 天** 历史数据（2016-01-26 至今）
  - 每日增量缓存到 `data/limitup_cache.json`
- **起始日对齐**: eIndex 起始日已调整为 2016-01-26，与 880006 最早数据对齐，三大指标从第一天起全部可用
