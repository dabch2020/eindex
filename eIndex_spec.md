# A股情绪指数 (eIndex) 项目开发规格文档

## 项目概述
eIndex是一个基于A股市场情绪指标的量化分析工具，通过计算全市场换手率、融资余额占比、涨停家数占比三个核心指标，生成综合情绪指数，为投资者提供客观的市场情绪参考。

## 核心策略

### 数据计算逻辑
每日收盘后（18:00后），待所有数据更新完毕，计算以下三个指标：

1. **全市场换手率** = 沪深两市总成交额 / 沪深两市总流通市值
2. **融资余额占比** = 融资余额 / 沪深两市总流通市值
3. **涨停家数占比** = 涨停股票数量 / 全市场股票总数

对每个指标计算其在过去250个交易日（约1年）中的**分位数**（0-100）：
- 分位数0：历史最低值
- 分位数100：历史最高值
- 当前分位数 = (当前值在历史数据中的排名位置) / 历史数据总数 × 100

**综合情绪指数** = (换手率分位数 + 融资余额占比分位数 + 涨停家数占比分位数) ÷ 3

### 交易信号规则
- **买入信号**：情绪指数 ≤ 20（市场极度恐慌）
- **卖出信号**：情绪指数 ≥ 80（市场极度狂热）
- **持有信号**：20 < 情绪指数 < 80（市场情绪正常）

### 数据源
- 腾讯财经API（实时性较好，免费）
- 东方财富网（备用）
- 聚宽/优矿（如需更精确数据，需API key）

### 指标卡片（实时显示）
- 当前情绪指数（大号数字，颜色渐变）
- 当前信号（买入/卖出/持有 + 图标）

### 今日三大指标值及分位数
- 交互图表（使用ECharts或Chart.js）
- 情绪指数历史走势图（可缩放时间轴）
- 数据从2015年开始
- 注意公共假日（A股不开盘的日期）不应该出现在统计中
- 买卖信号标记（在图表上标注信号点）
- 三大指标分位数对比图

### 数据表格
- 显示最近30个交易日数据
- 支持按日期、指数值、信号排序
- 提供CSV导出功能
